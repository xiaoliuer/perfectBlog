mysql和mongoDB的区别
发表于 2020-06-01| 更新于 2020-11-03|技术
阅读量:

mysql

1、为关系型数据库，在不同的引擎上有不同的存储方式

2、查询语句使用传统sql语句，有较为成熟的体系

3、强大的社区以及开发文档

mongodb

1、为非关系型数据库（文档型数据库）

2、使用自己独特的查询语句查询

3、存储方式是虚拟内存+持久化

4、属于新兴的nosql类型数据库，所以相对于mysql不够成熟

mongodb数据库的优势：

1、存储的数据为json数据格式，可以通过物理内存实现高速存取

2、自身failover机制，当自身的主库出现问题时，副本集会选择一个新的主库继续提供服务
